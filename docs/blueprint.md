# **App Name**: Presentation Pro

## Core Features:

- Real-time Clock Display: Displays the current time, configurable to AM/PM format.
- Data Input & Management: Allows input and editing of presentation metadata including Qualification, Component Code, Component Title, Start Time, End Time, and Extra Time.
- Single-Page Presentation: Displays all entered data on a single page in a clean, modern layout.
- Fullscreen Presentation Mode: A 'Start' button that triggers a fullscreen mode to present the information clearly.

## Style Guidelines:

- Primary color: Deep navy blue (#243A73) to convey professionalism and trustworthiness.
- Background color: Light gray (#E9ECEF) for a clean and modern aesthetic.
- Accent color: Soft teal (#6A8EAE) to provide visual interest without being distracting.
- Body and headline font: 'Inter', a sans-serif font known for readability and clean design.
- Use a responsive layout to ensure the presentation looks good on any screen size.
- Subtle fade-in animations when transitioning to fullscreen mode for a smooth user experience.
